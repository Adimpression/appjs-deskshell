To get the binaries for windows:-

1) Chrome
google chrome portable is available here: http://portableapps.com/apps/internet/google_chrome_portable
Chromium portable here: http://crportable.sourceforge.net/

Create a folder "chrome-profile" to store the chrome profile information in.

2) Node JS:
http://nodejs.org/download/ then select Windows Binary (.exe) 32-bit if you want the application to run on any windows computer. Save the binary to this folder.
The following node_modules are required: chrome-rdebug and request they can be installed using npm:

npm install chrome-rdebug
npm install request
